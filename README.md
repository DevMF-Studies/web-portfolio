## About This Project

This is my web portfolio built for my web development course. It includes a home page, a projects page, and a contact page. The site is designed to be responsive and accessible, using semantic HTML, and a Flexbox-based project gallery.

## What I Learned

I learned how to build a consistent layout across multiple pages using <header>, <nav>, <main>, and <footer>, along with headings, and tables. I learned how to create a responsive, card-based project gallery using Flexbox. I learned how to make layouts adapt to different screen sizes with media queries. I learned how to build an accessible contact form with labels and inputs, use alt text for images, and add visual polish with hover states, transitions, and a cohesive color theme.
